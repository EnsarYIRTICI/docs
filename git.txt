init -->
	git init
	git remote add origin https://github.com/EnsarYIRTICI/<repository-name>.git
	git add .
	git commit -m "first"
	git push origin master


update -->
	git add .
	git commit -m "bugfix"
	git push origin master


pull changes --> git pull


remote -->
	ls --> git remote -v
	add --> git remote add origin https://github.com/EnsarYIRTICI/<repository-name>.git
	change --> git remote set-url origin https://github.com/EnsarYIRTICI/<repository-name>.git


stop follow --> git rm -r --cached .


config -->
	small/large case --> git config core.ignorecase false

github -->
	 login -->
		sudo apt update
		sudo apt install gh
		gh auth login


feat: Yeni özellik
fix: Hata düzeltme

chore: Altyapı işleri (config, bağımlılık güncelleme, proje setup)
docs: Dokümantasyon değişiklikleri
style: Kod stil değişiklikleri (formatlama, boşluk, noktalama, linter uyumu)
refactor: Kodun davranışını değiştirmeden düzenleme
test: Test ekleme/düzenleme
perf: Performans iyileştirmeleri

build: Build sistemi veya bağımlılıklar ile ilgili değişiklikler (pubspec.yaml, gradle, package.json vs.)
ci: CI/CD yapılandırmaları veya script değişiklikleri (GitHub Actions, GitLab CI, Jenkins)

revert: Önceki bir commit’i geri alma
release: Versiyon artırma, changelog güncelleme (bazı ekipler chore(release) yazar)

merge: Branch birleştirmeleri (çoğu zaman otomatik üretilir)
deps: Bağımlılık güncellemeleri (chore yerine daha spesifik)
hotfix: Production ortamında acil düzeltme (bazı takımlar fix ile birleştiriyor)
security: Güvenlik açıklarını kapatma ile ilgili değişiklikler
design / ui: UI/UX değişiklikleri (renk, ikon, layout)
